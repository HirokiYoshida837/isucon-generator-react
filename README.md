# ISUCONジェネレーター.React

ISUCONのベンチーマーカや初期データの生成に利用されている処理のTypeScript/Reactに移植実装です。


## 内容

Next.js 13 の Page router を利用し、static exportしたものをGitHub Pageにて公開しています。


## 動かし方

```shell
npm run dev
```

## ビルド

```shell
npm run build
```
